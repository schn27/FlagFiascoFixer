# Flag Fiasco Fixer

Расширение для Chrome, которое восстанавливает отображение флага РФ на ресурсах LFM и WSS.

## Установка

Так как расширение не опубликовано в официальных магазинах расширений, установка производится вручную в режиме разработчика. Код открыт и доступен для проверки.

### Шаг 1: Подготовка файлов (для всех браузеров)

1. Нажмите кнопку **Code** -> **Download ZIP** в этом репозитории.
2. Распакуйте архив в любую удобную папку на компьютере (например, `C:\Extensions\flagfiascofixer`).
3. **Важно:** Не удаляйте эту папку после установки, иначе расширение перестанет работать.

### Шаг 2: Установка в браузер

Выберите инструкцию для вашего браузера:

#### Для Google Chrome

1. Введите в адресной строке `chrome://extensions/` и нажмите Enter.
2. В правом верхнем углу включите переключатель **Режим разработчика** (Developer mode).
3. Нажмите кнопку **Загрузить распакованное расширение** (Load unpacked).
4. Выберите папку, которую вы распаковали на Шаге 1.
5. Расширение появится в списке. Готово!

#### Для Microsoft Edge

1. Введите в адресной строке `edge://extensions/` и нажмите Enter.
2. В меню слева (или внизу страницы) включите переключатель **Режим разработчика** (Developer mode).
3. Нажмите кнопку **Загрузить распакованное** (Load unpacked).
4. Выберите папку, которую вы распаковали на Шаге 1.
5. Расширение появится в списке. Готово!

## Обновление

Поскольку расширение установлено вручную, оно **не обновляется автоматически**.

Чтобы получить новую версию:
1. Скачайте обновленные файлы из репозитория.
2. Замените файлы в папке расширения на компьютере.
3. На странице расширений (`chrome://extensions/` или `edge://extensions/`) нажмите кнопку **Обновить** (значок круговой стрелки) на карточке расширения.

## Важные примечания

- **Предупреждение браузера:** И Chrome, и Edge могут отображать сообщение *"Отключите расширения, работающие в режиме разработчика"* при каждом запуске. Это стандартное поведение безопасности для расширений, установленных не из магазина. Просто игнорируйте его или закрывайте крестиком.
- **Безопасность:** Код расширения открыт. Вы можете проверить файлы `manifest.json` и скрипты, чтобы убедиться в отсутствии вредоносной активности.
- **Совместимость:** Расширение тестировалось на последних версиях Google Chrome и Microsoft Edge.
